Jane Austen 스타일 분석

Author

정서윤 鄭㥠橍

N-gram 모델을 통해 배운 점

Jane Austen의 글은 N-gram 모델로 분석하면서, 우리는 언어가 반복의 규칙을 따른다는 것을 배웠다. 특히 “I must confess”, “It is certain” 같은 표현이 자주 등장했는데, 이는 감정의 명료한 표현이나 자기 주장을 나타낼 때 쓰이는 고유한 스타일이다.

데이터 전처리: Austen 문장의 구조화

austen <- austen_books() %>%
  group_by(book) %>%
  mutate(chapter = cumsum(str_detect(text, regex("^chapter [\\divxlc]", ignore_case = TRUE)))) %>%
  ungroup() %>%
  filter(!str_detect(text, "^\\s*$")) %>%
  filter(chapter != 0) %>%
  filter(!str_detect(text, "^((?i)CHAPTER\\s+[0-9IVXLCDM]+)$")) %>%
  mutate(text = gsub("[[:punct:]]", "", text))

오스틴 문체의 패턴적 특징

Austen 문장에서 발견된 특징은 다음과 같다:

  • 감정적 어휘의 반복: I must confess, I assure you, with all my heart
  • 논리적 연결어 선호: therefore, however, nevertheless
  • 자기 서술 표현 빈도: I must, It is
austen_uni <- austen %>%
  unnest_tokens(output = "unigram", input = text, token = "words")

austen_bi <- austen %>%
  unnest_tokens(output = "bigram", input = text, token = "ngrams", n = 2)

bi_count <- austen_bi %>%
  count(bigram, name = "bigram_count", sort = TRUE) %>%
  separate(bigram, c("w1", "w2"), sep = " ") %>%
  filter(if_any(c(w1, w2), ~ !is.na(.))) %>%
  filter(if_all(c(w1, w2), ~ !str_detect(., "^[0-9]")))

bi_probs <- bi_count %>%
  group_by(w1) %>%
  mutate(
    total_after_w1 = sum(bigram_count),
    probability = bigram_count / total_after_w1
  ) %>%
  ungroup() %>%
  arrange(desc(probability))

N-gram 기반 문장 생성 예시

generate_sentence <- function(bigram_probs, start_word = "still", max_len = 15) {
  sentence <- c(start_word)
  current_word <- start_word

  for (i in 1:max_len) {
    candidates <- bigram_probs[bigram_probs$w1 == current_word, ]
    if (nrow(candidates) == 0) break
    next_word <- sample(candidates$w2, size = 1, prob = candidates$probability)
    sentence <- c(sentence, next_word)
    current_word <- next_word
  }
  paste(sentence, collapse = " ")
}

set.seed(4567)
generate_sentence(bi_probs, start_word = "still", max_len = 15)
[1] "still it ought to go i shall only as second glance was not as to join"

Trigram을 통한 더 정교한 생성

joined <- paste(austen$text, collapse = " ")
austen_tri <- ngram(joined, 3)

set.seed(123)
babble(austen_tri, 20)
[1] "he replied I think her present home Elinors happiness was not so mean as to the event of that affection "

인사이트 요약

  • Jane Austen의 문체는 감성적 균형과 논리적 흐름을 모두 충족하는 특징이 있음
  • N-gram 모델을 활용하면 이러한 패턴을 수치화하고 문장 생성에도 응용 가능
  • 반복된 문장 구조는 단순한 문체가 아니라, 작가의 사고 구조와 가치관을 반영함
